About

  • July 2003

    The New Territory

    The parents decided to buy a house in Philadelphia so we moved down here. Starting new again.

    I am free!

    I graduated from Northeast High School ranked number 4 (woohoo!!) of 547. The parents were proud. I was glad I get to live by myself...until I had to do my own laundry, dishes and make my own food. Man, the things we take for granted.

Mario Game

I started this project in college for my Java class. I recently dug up this project and decided to put some final touches on it to make it more "complete". I developed it with Java AWT to paint sprites and background on the screen. The sprites move based on time and the user sprite (mario) changes direction based on user input. There's also collision detection to see if enemy sprite (bowser) and user sprite (mario) overlaps (collides). The hitbox is just a rectangle so nothing too fancy there. There are better ways for collision detection but I'll play around with them in other projects. Check it out here!
