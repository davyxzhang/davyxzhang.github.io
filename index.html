Mario Game

I started this project in college for my Java class. I recently dug up this project and decided to put some final touches on it to make it more "complete". I developed it with Java AWT to paint sprites and background on the screen. The sprites move based on time and the user sprite (mario) changes direction based on user input. There's also collision detection to see if enemy sprite (bowser) and user sprite (mario) overlaps (collides). The hitbox is just a rectangle so nothing too fancy there. There are better ways for collision detection but I'll play around with them in other projects. Check it out here!

Semantic Open Source Software (SEMOSS)

SEMOSS started as a Desktop application and has since evolved into a web application. I joined the project as a Java back-end developer in November 2013, and have moved to support the AngularJS front-end development.
SEMOSS is a powerful analytics tool and has been rapidly evolving to deliver intuitive UI and swift performance for large data sets. This application allows the users to effortlessly connect data from various different data sources, run analytic algorithms, transform data to D3 visualizations to discover insights, and then publish to share the insight with anyone. Check out these use cases to get a better idea of how SEMOSS is used.

Hackathon App: Connecting Small Businesses

This project was completed within a 24 hour period for a hackathon hosted by AngelHack. The goal of this hackathon was to come up with an idea and create an application that can solve a small business problems. Our group decided to create a multi-platform application that can help connect small businesses together. The goal is to allow small businesses to use this application to discover other small businesses that provide services that are needed.
The application was developed using the Ionic framework. We chose this framework because it allowed us to develop using AngularJS which is something we had experience with. Additionally, it uses Cordova to deploy to multiple mobile platforms (iOS & Android). We used Java to develop our backend and used the Yelp API to pull data about local small businesses.
After 24 hours, we were successfully able to finish the prototype of the application. We were all proud and surprised about how much we had accomplished in such a short amount of time. Unfortunately, we focused too much on the technical pieces of our idea that we did not have enough time to come up with a good pitch. Nevertheless, it was a fun and exciting experience and I learned much from it. Check out the github page for our work here.

Angular Auto-complete: Excel Style

This is an AngularJS directive based off of Daryl Rowland's angucomplete. The customizations were inspired by Excel's Formula AutoComplete. I was not able to find existing work that could do what I needed, so I decided to add my own. I needed it for completing a functionality to derive a new column based on existing data, similar to how Excel does it. You can check out my example here. I used Math.js to parse and evaluate a formula and then add the derived data to my existing dataset. I have some ideas to improve the autocomplete feature and will get to those as time permits.
